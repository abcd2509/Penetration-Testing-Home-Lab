# Enterprise Vulnerability & Penetration Testing Lab

## Project Overview
An isolated, virtualized home lab environment built to safely conduct vulnerability assessments, network enumeration, and penetration testing methodologies. 

## Lab Architecture
* **Hypervisor:** VirtualBox
* **Attacker Machine:** Kali Linux
* **Target Machine:** Metasploitable 2
* **Network Isolation:** Internal Network (Isolated virtual segment; zero host-to-guest or guest-to-host routing).
